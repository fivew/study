
# 复制&粘贴
- y10j	复制10行    
- 10jp	在10后面粘贴	
- crtl + v/shift + v	选择区域复制   
- yy	复制游标所在行整行。或大写一个Y。 
- 2yy	或y2y复制两行
- y^	复制至行首，或y0。不含游标所在处字元。 
- y$	复制至行尾。含游标所在处字元。 
- yw 	复制一个word。 
- y2w	复制两个字（单词）。 
- yG 	复制至档尾。 
- y1G	复制至档首。 
- p 	小写p代表贴至游标后（下）。 
- P 	大写P代表贴至游标前（上）。
- 跨文本复制粘贴 vim有效
  - 选中复制文本 +y
  - 粘贴 +p

-set paste　原格式拷贝，不会变形

# 快捷键
  u： 撤销
  ctrl + r : 重做
